Relative, Absolute & Fixed Positioning

HOME

Relative Positioning

Positions an element relative to it's original position. Meaning, the element is still in the normal document flow and takes up space like any other element. This element still affects the layout of other elements. The position is specified by the top, right, bottom or left properties.

Absolute Positioning

Positions an element relative to it's nearest positioned ancestor. This removes an an element from the normal document flow, meaning it won't affect the layout of other elements.

Fixed Positioning

Positions an element rleative to the visible area of the browser window (viewport), rather than the containing blow or parent element. The element stays in the same position even when the page is scrolled.
