Describe what the DOM is and an example of how you might interact with it.

The DOM = Document Object Model, is a way of accessing files in the html from your javascript. Here you can access elements by id's, classes and more and change things within it. You use this in conjuction with functions to return specific results. For example, if you run out of stock on the object - turn the text red. It's crucial with responsive web design.

Explain the difference between accessing data from arrays and objects

Accessing data from arrays means that you are accessing potentially multiple pieces of information. While accessing data from objects requires you to use a named key to retrieve a specifc value. Arrays are best for ordered list (names) and objects are better for representing names properties of single entity. (products on a menu)
